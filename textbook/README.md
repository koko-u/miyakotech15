# はじめに

## サーバーの環境

本日のサーバー環境はAWS上に構築しています。

| 項目               | 値                          |
| ----               | ----                        |
| インスタンスタイプ | t2.small(vCPUx1, メモリ2GB) |
| OS                 | Ubuntu 18.04                |
| 開放ポート         | 22, 80, 443

- ライセンスはMITとしています（再利用等はご自由にどうぞ）
